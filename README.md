# Sales_Performance_Dashboards | Power BI
End-to-end sales analytics dashboard with KPI tracking and insights.

## Project Overview
This project is an interactive Sales Performance Dashboard built using Power BI to analyze sales trends, target achievement, variance, and salesperson performance.

The dashboard helps businesses track KPIs, compare actual sales against targets, and identify performance trends for better decision-making.

## Business Problem

Businesses often struggle to monitor whether sales teams are meeting their targets consistently. This dashboard was created to analyze sales performance, identify gaps between actual and target sales, and help stakeholders make data-driven decisions.

The dashboard enables:
- Sales trend analysis
- Target vs Actual comparison
- Performance monitoring
- Variance analysis
- Salesperson-level insights

## Key KPIs

- Total Actual Sales
- Total Target Sales
- Variance
- Variance %
- Monthly Target Reached
- Salesperson Performance Analysis
- Target Status

  ## Tools & Technologies Used

- Power BI
- DAX
- Power Query
- Data Modeling
- Microsoft Excel

## Dashboard Features

- Interactive dashboard design
- Monthly Actual vs Target sales comparison
- Variance and variance percentage analysis
- Salesperson-level performance tracking
- Dynamic filtering using slicers
- Trend analysis and KPI monitoring
- Business insights using visual storytelling
